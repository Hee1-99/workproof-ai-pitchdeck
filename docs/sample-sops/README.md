# WorkProof Sample SOP Set

이 폴더는 WorkProof MVP/데모에서 사용할 수 있는 공개자료 기반 샘플 SOP 세트입니다.

주의:
- 아래 문서들은 특정 회사의 실제 내부 SOP가 아닙니다.
- KOSHA/고용노동부 등 공공 안전자료와 일반적인 현장 안전수칙을 바탕으로 MVP 테스트용으로 재구성한 “SOP-like” 문서입니다.
- 실제 현장 적용 전에는 사업장 안전관리자/관리감독자의 검토가 필요합니다.
- WorkProof의 목적은 법정 안전교육 대체가 아니라, 사업장별 작업지시 이해도 확인과 교육 기록 보조입니다.

## Files

1. `forklift_precheck_ko.md` — 지게차 작업 전 점검 및 이동 작업
2. `press_cleaning_lockout_ko.md` — 프레스 설비 청소/점검 전 전원차단
3. `welding_fire_prevention_ko.md` — 용접·용단 화재 예방 작업
4. `confined_space_entry_ko.md` — 밀폐공간 출입 전 안전 확인
5. `conveyor_jam_clear_ko.md` — 컨베이어 끼임/걸림 해소 작업

## Recommended demo flow

1. 샘플 SOP 1개를 WorkProof에 업로드한다.
2. AI가 작업 단계, 위험요소, 보호구, 금지행동을 추출한다.
3. 베트남어/네팔어/캄보디아어 등으로 교육 카드를 생성한다.
4. 작업자가 QR로 접속해 교육을 완료한다.
5. 관리자 대시보드에서 완료 시간, 언어, 퀴즈 점수, 서명 로그를 확인한다.
