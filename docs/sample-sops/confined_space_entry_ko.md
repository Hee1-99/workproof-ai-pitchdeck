# 샘플 SOP: 밀폐공간 출입 전 안전 확인

## 문서 목적
탱크, 맨홀, 피트, 저장조 등 밀폐공간 작업 중 질식·중독 사고를 예방하기 위한 출입 전 확인 절차를 정의한다.

## 적용 범위
- 산소결핍 또는 유해가스 위험이 있는 밀폐공간
- 점검, 청소, 보수, 내부 확인 작업

## 필요한 보호구 및 장비
- 산소/가스 농도 측정기
- 환기장치
- 안전대 및 구명줄
- 안전모
- 보호장갑
- 필요 시 송기마스크 또는 공기호흡기

## 작업 전 확인
1. 해당 장소가 밀폐공간인지 확인한다.
2. 작업 허가서 또는 관리감독자 승인을 받는다.
3. 산소 농도와 유해가스 농도를 측정한다.
4. 산소 농도가 안전 범위인지 확인한다.
5. 환기장치를 가동하여 내부 공기를 충분히 교체한다.
6. 감시자를 외부에 배치하고 연락 방법을 정한다.
7. 구조 장비와 비상 연락망을 준비한다.

## 작업 절차
1. 작업자는 보호구를 착용하고 출입 전 최종 승인을 받는다.
2. 출입 전 산소/가스 농도 측정 결과를 기록한다.
3. 환기를 유지한 상태에서 작업한다.
4. 감시자는 작업자가 내부에 있는 동안 자리를 비우지 않는다.
5. 작업자가 어지러움, 두통, 호흡곤란을 느끼면 즉시 작업을 중지하고 대피한다.
6. 구조가 필요한 경우 보호장비 없이 내부로 들어가지 않는다.
7. 작업 완료 후 인원과 장비를 확인하고 출입구를 정리한다.

## 금지 행동
- 농도 측정 없이 밀폐공간 출입 금지
- 환기 없이 작업 금지
- 감시자 없이 단독 작업 금지
- 보호장비 없이 구조하러 진입 금지

## 주요 위험요소
- 산소결핍 질식
- 황화수소 등 유해가스 중독
- 구조 중 2차 사고
- 내부 추락 또는 미끄러짐

## 이해도 확인 퀴즈 예시
1. 밀폐공간 출입 전 반드시 해야 하는 것은 무엇인가?
   - A. 산소/가스 농도 측정
   - B. 작업복 색상 확인
   - C. 휴대전화 충전
2. 동료가 쓰러졌을 때 보호장비 없이 바로 들어가도 된다. (O/X)
3. 작업 중 환기는 계속 유지해야 한다. (O/X)
