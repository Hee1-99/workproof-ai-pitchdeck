# 샘플 SOP: 프레스 설비 청소/점검 전 전원차단

## 문서 목적
프레스 설비 청소, 점검, 금형 주변 이물 제거 작업 중 끼임·협착 사고를 예방하기 위한 안전 절차를 정의한다.

## 적용 범위
- 프레스 설비 청소
- 금형 주변 이물 제거
- 설비 점검 전 준비 작업

## 필요한 보호구
- 안전모
- 보안경
- 절단방지 장갑
- 안전화

## 작업 전 확인
1. 작업자는 설비 담당자에게 청소/점검 작업 시작을 알린다.
2. 설비가 완전히 정지했는지 확인한다.
3. 주 전원을 차단하고, 조작반에 “작업 중” 표시를 부착한다.
4. 가능하면 잠금장치와 태그를 사용하여 다른 사람이 전원을 켜지 못하게 한다.
5. 잔류 압력, 잔류 에너지, 관성 회전이 없는지 확인한다.
6. 2인 1조 작업이 필요한 경우 감시자를 배치한다.

## 작업 절차
1. 전원 차단 후 조작 버튼을 눌러 설비가 작동하지 않는지 확인한다.
2. 금형 사이 또는 위험구역에 손을 넣기 전 담당자 확인을 받는다.
3. 이물 제거는 손이 아닌 전용 도구를 사용한다.
4. 청소 중 다른 작업자가 조작반에 접근하지 못하게 한다.
5. 작업 완료 후 사용한 공구와 이물질을 제거한다.
6. 보호구 착용 상태와 작업구역 정리 상태를 확인한다.
7. 담당자 승인 후 “작업 중” 표시를 제거하고 전원을 복구한다.
8. 시험 운전 시 작업자가 위험구역 밖에 있는지 확인한다.

## 금지 행동
- 전원 차단 없이 금형 사이에 손 넣기 금지
- 혼자 판단하여 임의로 전원 복구 금지
- 손으로 직접 이물 제거 금지
- 방호장치 해제 상태로 운전 금지

## 주요 위험요소
- 금형 사이 끼임
- 갑작스러운 설비 작동
- 잔류 에너지에 의한 협착
- 이물 제거 중 손 베임

   - B. 전원 차단과 설비 정지
   - C. 생산 수량
