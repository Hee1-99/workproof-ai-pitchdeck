# 샘플 SOP: 용접·용단 화재 예방 작업

## 문서 목적
용접·용단 작업 중 불티, 고온 금속, 가연물 접촉으로 인한 화재와 폭발을 예방하기 위한 절차를 정의한다.

## 적용 범위
- 제조공장, 조선소, 건설현장 등 용접·용단 작업
- 화기 작업자, 감시자, 관리감독자

## 필요한 보호구
- 용접면 또는 보안면
- 방염 장갑
- 방염 앞치마 또는 작업복
- 안전화
- 필요 시 방진마스크

## 작업 전 확인
1. 화기작업 허가가 필요한 현장인지 확인한다.
2. 작업구역 주변의 종이, 비닐, 목재, 기름, 스프레이 등 가연물을 제거한다.
3. 제거가 어려운 가연물은 불연성 덮개로 보호한다.
4. 소화기와 소화수를 작업구역 가까이에 배치한다.
5. 밀폐공간 또는 탱크 내부 작업은 가스 농도와 산소 농도를 확인한다.
6. 필요 시 화재감시자를 배치한다.

## 작업 절차
1. 작업자는 보호구를 착용하고 주변 작업자에게 화기작업 시작을 알린다.
2. 불티가 떨어지는 방향과 아래층/틈새 공간을 확인한다.
3. 용접 중 가연물 방향으로 불티가 튀지 않도록 차단막을 설치한다.
4. 작업 중 연기, 냄새, 불꽃이 비정상적으로 발생하면 즉시 중지한다.
5. 작업 완료 후 최소 지정 시간 동안 잔불을 확인한다.
6. 작업구역 바닥, 틈새, 주변 적재물에 불씨가 남아 있지 않은지 확인한다.
7. 이상이 없으면 담당자에게 작업 종료를 보고한다.

## 금지 행동
- 가연물 제거 없이 용접 시작 금지
- 소화기 없이 화기작업 금지
- 밀폐공간에서 가스 농도 확인 없이 작업 금지
- 작업 완료 직후 현장 이탈 금지

## 주요 위험요소
- 불티에 의한 화재
- 인화성 증기 폭발
- 유해 흄 흡입
- 고온 금속에 의한 화상

## 이해도 확인 퀴즈 예시
1. 용접 전 주변 가연물은 어떻게 해야 하는가?
   - A. 그대로 둔다
   - B. 제거하거나 불연성 덮개로 보호한다
   - C. 물건을 더 쌓아둔다
2. 소화기가 없으면 짧은 용접 작업은 해도 된다. (O/X)
3. 작업 완료 후 잔불 확인이 필요한 이유는 무엇인가?
