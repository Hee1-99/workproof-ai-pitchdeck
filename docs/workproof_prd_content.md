# WorkProof PRD

## 0. 문서 개요
- 제품명: WorkProof
- 한줄 정의: 외국인 근로자가 우리 사업장의 작업지시서를 모국어로 학습하고, 실제로 이해했음을 관리자에게 증빙해주는 안전 온보딩 도구.
- 작성 목적: 해커톤/MVP 개발을 위한 제품 요구사항 정의와 시장성 근거 정리.
- 핵심 관점: WorkProof는 “SOP 번역기”가 아니라 “교육 리스크와 이해도 증빙”을 해결하는 B2B SaaS다.

## 1. Executive Summary
한국의 제조·건설·조선·물류 현장은 외국인 근로자 의존도가 높아지고 있다. 동시에 안전교육과 작업지시는 여전히 한국어 중심이거나, 공통 안전자료·수동 번역·통역 가능한 직원에게 의존한다. 문제는 교육자료가 없다는 것이 아니라, “우리 사업장의 실제 작업지시를 근로자가 이해했는지”를 확인하고 기록하기 어렵다는 점이다.

WorkProof는 사업장별 SOP/PDF/작업표준서를 업로드하면 AI가 작업 단계, 위험요소, 보호구, 금지행동을 추출하고, 이를 다국어 모바일 교육 카드와 짧은 이해도 퀴즈로 변환한다. 근로자는 QR로 접속해 모국어로 교육을 받고, 관리자는 근로자별 완료 시간, 언어, 퀴즈 결과, 전자서명 로그를 보관한다.

초기 시장은 경기·인천·충남의 외국인 근로자 고용 중소 제조업과 조선업 협력사, 식품·포장·물류센터다. 초기 MVP는 완전 자동화보다 컨시어지 방식으로 실제 SOP 3~10개를 교육 카드로 변환하고, 유료 파일럿/LOI 확보를 목표로 한다.

## 2. 시장·문제 리서치 요약
### 2.1 국내 외국인 근로자 산재와 언어 문제
1. 고용노동부/안전보건공단은 외국인 근로자의 증가와 언어문제에 따른 산업재해 우려를 배경으로, 산재 다발 작업에 대한 외국인 근로자용 안전보건 교육자료를 제작·배포했다. 용접작업 화재, 밀폐공간 질식 등 재해예방 교육 동영상을 16개국 언어로 번역했고, 누적 외국인 안전보건 콘텐츠는 약 1,500종으로 언급된다.
2. KBS 보도 역시 “외국인 근로자의 언어문제에 따른 산업재해를 예방하기 위해” 정부가 16개국 언어 안전보건 자료를 배포했다고 보도했다.
3. 이민정책연구원 보고서 관련 연합뉴스 보도에 따르면 2023년 외국인 근로자 업무상 사고 산재 승인 8,434건 중 78.1%가 건설업과 광·제조업에서 발생했다. 세부적으로 건설업 39.8%, 광·제조업 38.3%다.
4. 조선비즈는 “외국인 근로자 잇단 사고… 의사소통 안돼 안전 교육 부실”이라는 프레임으로, 한국어에 서툰 외국인 근로자가 업무 지시나 안전수칙을 제대로 이해하지 못했을 가능성 및 현장 안전교육 부실 문제를 다뤘다.
5. 경기도는 언어 장벽과 교육 접근성 부족으로 산업재해 위험에 노출된 외국인 노동자를 위해 VR·AI 기반 안전교육을 추진한다는 보도가 있었다. 이는 지자체 차원에서도 외국인 근로자 안전교육 문제가 정책 의제로 다뤄지고 있음을 보여준다.

### 2.2 해외/비교 사례
1. OSHA는 안전교육이 근로자가 이해할 수 있는 언어와 어휘로 제공되어야 한다는 정책을 명시한다. 이는 미국 시장 확장 시 compliance positioning의 근거가 된다.
2. Squint의 공개 case study에서는 글로벌 산업/에너지 기업이 다국어 유지보수 절차를 시각적·단계별로 제공한 뒤 language-related error가 98% 감소하고, 18개월간 4,200시간 이상의 downtime을 방지했다고 주장한다.
3. 이 사례는 “다국어 작업지시” pain이 존재함을 보여주지만, Squint는 엔터프라이즈 산업 플랫폼에 가깝다. WorkProof는 한국 중소 사업장의 외국인 온보딩과 이해도 증빙이라는 좁은 wedge로 시작한다.

## 3. 문제 정의
### 3.1 핵심 문제
사업주는 안전교육을 했다고 생각하지만, 외국인 근로자가 실제로 작업지시와 위험요소를 이해했는지 확인하고 증빙하기 어렵다.

### 3.2 현재 대안과 한계
- 공공 안전자료: 공통 위험 안내에는 유용하지만, 사업장별 실제 설비·공정·SOP를 반영하기 어렵다.
- 수동 번역: 품질 편차가 크고, 작업 단계·위험요소·금지행동 구조화가 어렵다.
- 통역 가능한 직원 의존: 특정 직원에게 지식이 집중되고, 교육 기록이 남지 않는다.
- 종이 서명부: 교육 참석 기록은 남지만 이해도 확인과 언어별 교육 내용 증빙이 약하다.
- 범용 번역기: 문장은 번역하지만 교육 플로우, 퀴즈, 서명, 재교육 로그를 제공하지 않는다.

### 3.3 해결해야 할 핵심 질문
- 이 근로자는 어떤 작업을 어떤 언어로 배웠는가?
- 위험요소와 금지행동을 실제로 이해했는가?
- 사고/감사/원청 대응 시 보여줄 기록이 있는가?
- 신규 외국인 근로자 투입 시 교육 시간을 줄일 수 있는가?

## 4. 타깃 사용자와 구매자
### 4.1 초기 타깃 세그먼트
- 경기·인천·충남 중소 제조업
- 조선업 협력사
- 식품·포장 공장
- 물류센터
- 외국인 고용 비중이 높은 뿌리산업: 용접, 표면처리, 금형, 주조 등

### 4.2 사용자/구매자 구분
- 실제 사용자: 외국인 근로자, 현장 반장, 신규 입사자
- 관리자 사용자: 안전관리자, 공장장, 생산관리자, 노무 담당자
- 구매자: 중소기업 대표, 공장장, 안전/노무 책임자
- 영향자: 원청 안전 담당자, 노무사, 산업안전 컨설턴트, 지자체/지원기관

### 4.3 Trigger Moment
- 신규 외국인 근로자 입사 또는 공정 배치
- 산재 사고/아차사고 발생 후 재교육 필요
- 원청/감사에서 교육 기록 요구
- 외국인 근로자 증가로 기존 구두 교육이 한계에 도달
- 통역 가능한 직원 퇴사 또는 부재

## 5. 제품 포지셔닝
### 5.1 한줄 포지셔닝
외국인 근로자가 우리 공장의 작업지시를 모국어로 이해하고, 관리자가 그 이해 여부를 증빙할 수 있게 하는 AI 안전 온보딩 도구.

### 5.2 하지 말아야 할 포지셔닝
- AI SOP 번역기
- 다국어 문서 생성기
- 대기업용 AR 현장 플랫폼
- LMS 전체 대체재

### 5.3 해야 할 포지셔닝
- 외국인 근로자 안전교육 증빙
- 사업장별 SOP 이해도 확인
- 신규 투입 전 온보딩 리스크 감소
- 사고/감사/원청 대응용 교육 로그

## 6. MVP 범위
### 6.1 MVP 목표
2주 내 실제 사업장 SOP를 교육 카드로 변환하고, 외국인 근로자 5~10명이 모국어로 교육을 완료하며, 관리자에게 이해도 로그를 보여주는 데모/파일럿을 만든다.

### 6.2 핵심 사용자 플로우
1. 관리자가 SOP/PDF/작업표준서를 업로드한다.
2. 시스템이 작업 단계, 위험요소, 보호구, 금지행동을 추출한다.
3. 관리자가 추출 결과를 검토/수정한다.
4. 시스템이 선택 언어별 교육 카드를 생성한다.
5. 작업자가 QR 링크로 접속한다.
6. 작업자는 모국어로 단계별 카드를 확인한다.
7. 작업자는 3~5문항 이해도 퀴즈를 푼다.
8. 통과 시 전자서명 또는 확인 버튼을 누른다.
9. 관리자는 완료 로그, 점수, 언어, 시간, 서명을 확인한다.
10. 미통과자는 재교육 대상으로 표시된다.

### 6.3 Must-have 기능
- 관리자 로그인 또는 간단한 사업장 코드 접근
- SOP/PDF 텍스트 업로드
- 작업 단계/위험요소/보호구/금지행동 추출
- 다국어 교육 카드 생성: 1차 언어 후보는 베트남어, 네팔어, 캄보디아어, 태국어, 영어, 중국어
- 작업자용 모바일 화면
- 이해도 퀴즈 생성/응답
- 완료 로그: 근로자명 또는 식별번호, 언어, 작업명, 완료 시간, 점수, 서명 여부
- 관리자 대시보드: 교육 완료/미완료/재교육 필요 상태

### 6.4 Nice-to-have 기능
- 음성 읽어주기
- 이미지/사진 기반 단계 설명
- 교육 만료/재교육 알림
- 작업별 QR 포스터 출력
- 원청/감사용 PDF 리포트 export
- 근로자별 교육 이력 카드

### 6.5 Non-goals
- 실시간 영상 분석
- ERP/MES 대규모 연동
- 완전한 LMS 대체
- 법적 안전교육 이수 인증 대체 주장
- 의료/개인 민감정보 처리
- 모든 언어의 완벽한 전문 번역 보장

## 7. 상세 요구사항
### 7.1 관리자 기능
- 사업장 생성 및 작업지시서 등록
- 문서 업로드: PDF, 이미지, 텍스트. 해커톤 MVP에서는 PDF/텍스트 우선
- AI 추출 결과 편집
- 교육 언어 선택
- 작업자 초대 링크/QR 생성
- 교육 완료 현황 확인
- 개별 작업자 교육 로그 조회
- CSV 또는 PDF 형태의 증빙 리포트 다운로드

### 7.2 작업자 기능
- 별도 회원가입 없이 QR/링크 접속
- 언어 선택 또는 관리자 지정 언어 자동 적용
- 단계별 교육 카드 열람
- 위험/금지 행동 강조 카드 확인
- 이해도 퀴즈 응답
- 전자서명 또는 확인 체크
- 미통과 시 다시 보기/재시도

### 7.3 AI 기능
- SOP에서 작업명, 목적, 사전 준비, 단계, 위험요소, PPE, 금지행동, 응급 대응 문구 추출
- 쉬운 한국어 요약 생성
- 다국어 번역 생성
- 퀴즈 문항 생성: O/X, 객관식, 순서 맞추기
- 관리자 검수 전에는 작업자에게 공개하지 않음

### 7.4 보안/컴플라이언스 기본 원칙
- 사업장 문서와 작업자 로그는 사업장별로 분리
- 공개 링크는 토큰 기반으로 접근 제한
- 작업자 개인정보는 최소 수집: 이름 대신 사번/별칭 허용
- 관리자 검수 없는 AI 결과 자동 배포 금지
- 교육 완료 기록은 수정 이력 또는 재발급 이력을 남기는 방향 권장

## 8. 데이터 모델 초안
### 8.1 Entities
- Organization: id, name, industry, region, admin_user_id
- User: id, org_id, role, name, email/phone optional
- Worker: id, org_id, display_name, language, worker_code
- SOPDocument: id, org_id, title, file_url, source_text, created_by, created_at
- TrainingModule: id, org_id, sop_document_id, title, status, languages, reviewer_id
- TrainingStep: id, module_id, order, title, instruction, hazard, ppe, prohibited_action
- QuizQuestion: id, module_id, language, question, options, answer, explanation
- Assignment: id, module_id, worker_id, due_at, status
- CompletionLog: id, assignment_id, language, score, passed, signed_at, completed_at, signature_hash

### 8.2 주요 상태값
- TrainingModule.status: draft, reviewed, published, archived
- Assignment.status: not_started, in_progress, passed, failed, expired

## 9. 성공 지표
### 9.1 MVP 검증 지표
- 인터뷰 15명 중 5명 이상이 현재 교육 기록/언어 문제를 명확한 pain으로 인정
- 실제 SOP 3개 이상 확보
- 외국인 근로자 5~10명 테스트 완료
- 관리자 3곳 이상이 유료 파일럿 또는 LOI 동의
- 교육 카드 1개 생성 소요 시간 30분 이하로 감소

### 9.2 제품 지표
- 교육 완료율
- 퀴즈 통과율
- 재교육 필요 비율
- 신규 근로자 온보딩 소요 시간
- 사업장별 월 활성 교육 모듈 수
- 유료 전환율
- 월 반복 매출

### 9.3 Kill Criteria
- 관리자들이 “좋다”라고 말하지만 교육 기록/감사/사고 대응에 돈을 내지 않음
- 실제 SOP 제공을 꺼려 파일럿 진행이 어려움
- 외국인 근로자가 모바일 교육 카드보다 대면 통역을 선호하고 완료율이 낮음
- 법적 인증 대체를 요구하지만 제품 범위에서 제공할 수 없음

## 10. 2주 검증 계획
### D1-D3: 고객 인터뷰
대상: 공장장, 안전관리자, 노무 담당자, 노무사, 산업안전 컨설턴트 15명.
질문:
- 외국인 근로자 신규 투입 시 교육은 어떻게 하나요?
- 교육 자료는 어떤 언어로 제공되나요?
- 작업자가 실제로 이해했는지 어떻게 확인하나요?
- 사고/감사/원청 대응 때 교육 기록을 요구받은 적이 있나요?
- 통역/번역/교육 기록에 월 얼마의 비용 또는 시간이 들어가나요?
- 이런 도구가 있으면 월 10만~30만원을 낼 이유가 있나요?

### D4-D7: 컨시어지 MVP
- 실제 SOP 3개 확보
- 수동+AI로 교육 카드/퀴즈/서명 플로우 제작
- 베트남어/네팔어/캄보디아어 중 최소 2개 언어 지원

### D8-D10: 현장 테스트
- 외국인 근로자 5~10명 테스트
- 이해도, 번역 자연스러움, 카드 길이, 퀴즈 난이도 피드백 수집

### D11-D14: 지불 검증
- 유료 파일럿 제안: 2주 10만~30만원
- LOI 또는 파일럿 동의 3곳 목표
- 실패 시 타깃 업종 또는 포지셔닝 조정

## 12. 경쟁/대안 분석
### 12.1 Squint
- 강점: 엔터프라이즈 산업 현장 지식 관리, 작업 지시, AI assistant, analytics.
- 약점/틈: 대기업/글로벌 사업장 중심으로 보이며, 한국 중소 제조업의 빠른 QR 기반 교육 증빙 니즈에는 과할 수 있음.
- WorkProof 차별점: 한국 사업장, 외국인 신규 투입, 이해도 퀴즈/서명 로그, 낮은 도입 비용.

### 12.2 공공 안전보건 자료
- 강점: 무료, 다국어, 공신력.
- 약점: 사업장별 SOP 반영 부족, 근로자별 이해도 증빙 부족.
- WorkProof 차별점: 우리 공장 문서 기반 맞춤형 교육과 로그.

### 12.3 범용 번역기/ChatGPT
- 강점: 빠른 번역과 요약.
- 약점: 교육 완료 관리, 퀴즈, 서명, 로그, 감사 대응 리포트 부재.
- WorkProof 차별점: 워크플로우와 증빙.

## 13. 리스크와 대응
- 법적 인증 오해: “법정 교육 대체”가 아니라 “사업장별 이해도 확인과 기록 보조”로 명확히 표기.
- 번역 오류: 관리자 검수 단계 필수, 중요 위험 문구는 원문 병기.
- 개인정보: 이름 대신 사번/코드 사용 가능, 최소 수집 원칙.
- 현장 디지털 접근성: QR 기반, 별도 앱 설치 없는 모바일 웹 우선.
- 영세 사업장 지불 의사 부족: 초기에는 문서 세팅비/컨설팅형 유료 파일럿으로 검증.
- 콘텐츠 책임: AI 생성 결과를 최종 안전 지침으로 단독 사용하지 않도록 리뷰 플로우 제공.

## 16. SOP 자료 확보 전략 및 샘플 SOP 세트
### 16.1 실제 현장 SOP 확보 가능성
실제 회사 내부 SOP 원본은 보통 영업비밀, 설비 정보, 안전관리 체계가 포함되어 있어 공개적으로 구하기 어렵다. 따라서 해커톤/MVP 단계에서는 공개 안전자료와 표준작업절차서 양식을 기반으로 “SOP-like” 샘플을 만들고, 이후 파일럿 인터뷰에서 익명화된 실제 문서 일부를 확보하는 방식이 현실적이다.

### 16.2 MVP 단계에서 사용할 수 있는 자료 유형
- 고용노동부/안전보건공단의 외국인 근로자용 안전보건 교육자료
- KOSHA 자료 아카이브의 작업별 안전수칙, OPS, 안전작업 안내문
- 지게차, 컨베이어, 프레스, 용접, 밀폐공간 등 산재 다발 작업 자료
- 제조업 작업표준서/안전작업절차서 공개 양식
- 회사명/제품명/수치가 제거된 익명화 실제 작업지시서 일부

### 16.3 WorkProof repo에 추가한 샘플 SOP
다음 샘플 SOP는 특정 회사의 실제 내부 문서가 아니라, 공공 안전자료와 일반적인 현장 안전수칙을 바탕으로 MVP 데모용으로 재구성한 문서다. 실제 현장 적용 전에는 반드시 사업장 안전관리자/관리감독자의 검토가 필요하다.

- `docs/sample-sops/forklift_precheck_ko.md` — 지게차 작업 전 점검 및 이동 작업
- `docs/sample-sops/press_cleaning_lockout_ko.md` — 프레스 설비 청소/점검 전 전원차단
- `docs/sample-sops/welding_fire_prevention_ko.md` — 용접·용단 화재 예방 작업
- `docs/sample-sops/confined_space_entry_ko.md` — 밀폐공간 출입 전 안전 확인
- `docs/sample-sops/conveyor_jam_clear_ko.md` — 컨베이어 끼임/걸림 해소 작업

### 16.4 실제 SOP 확보 요청 문구
“실제 공정명이나 회사명은 전부 가려도 됩니다. 외국인 근로자 교육용으로 작업지시서를 다국어 카드와 이해도 퀴즈로 바꾸는 MVP를 만들고 있어서, 지게차/프레스/용접/세척 같은 단순 작업 SOP 샘플 1개만 테스트해보고 싶습니다.”

### 16.5 샘플 SOP 활용 방법
1. 샘플 SOP 1개를 WorkProof에 업로드한다.
2. AI가 작업 단계, 위험요소, 보호구, 금지행동을 추출한다.
3. 베트남어/네팔어/캄보디아어 등으로 교육 카드를 생성한다.
4. 작업자가 QR로 접속해 교육을 완료한다.
5. 관리자 대시보드에서 완료 시간, 언어, 퀴즈 점수, 서명 로그를 확인한다.

### 16.6 샘플 SOP 관련 리스크
- 샘플 SOP는 법적 안전교육 또는 공식 작업허가서를 대체하지 않는다.
- 공공자료 기반으로 재구성한 문서이므로 현장별 설비, 공정, 위험성평가 결과를 반영해야 한다.
- 투자자/심사위원 데모에서는 “실제 내부 SOP”가 아니라 “공개자료 기반 데모용 SOP”임을 명확히 표시한다.
